## 1. One-Line Summary

这是一篇分析型论文，把 KV compression 理解为对 token-level routing 的受控扰动，并试图解释为什么某些压缩比例下会突然出现幻觉与可达性崩塌。

## 2. Why It Matters

它不直接提出新的压缩算法，但它解释了为什么“保留 token”并不等于“还能访问语义”。这对评估 KV compression 尤其重要，因为很多论文只看 benchmark accuracy，而忽略 routing dynamics 的结构性损伤。

## 3. Main Idea

- 提出一个 physics-inspired 视角，把 KV compression 看作对 attention routing 的扰动，而不仅仅是存储裁剪。
- 区分 `retention`、`accessibility`、`utilization` 三个层次，强调 token 还在 cache 里不代表语义仍然可达。
- 用 synthetic tasks 分析 multi-entity tracking、disambiguation、coreference、multi-hop reasoning 等场景。
- 观察到中等压缩下内部表征已恶化但准确率暂时不掉，说明存在冗余。
- 观察到接近 `90%` 压缩时出现 hallucination safety cliff，并与 Global Eviction Ratio 上升相关。

## 4. Relation to Existing Work

- 相比 H2O / SnapKV / KVSlimmer，这篇不是方法论文，而是解释这些方法可能在哪些 regime 失效。
- 相比一般 benchmark paper，它更强调 routing geometry 与语义可达性，而不是只报 LongBench 分数。
- 适合作为 repo 中 `analysis / understanding` 类型的补充材料，但不是主干算法条目。

## 5. Evidence

- 摘要没有给出统一的吞吐或内存数字，而是报告现象级发现。
- 核心证据是: moderate compression 下 representation 已退化；接近 `90%` 压缩出现安全 cliff；不同模型的 routing dynamics 不同，表现出不同压缩韧性。
